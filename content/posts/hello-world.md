### なぜこの構成？

> 「シンプルで、速くて、維持しやすい」

これが選定の基準でした。

**Hono** は超軽量なWebフレームワークで、Cloudflare Workersとの相性が抜群です。Express.jsライクなAPIなので学習コストも低く、JSXでUIを書けるのも魅力的でした。

```typescript
import { Hono } from 'hono'

const app = new Hono()

app.get('/', (c) => c.text('Hello World!'))

export default app
```

たったこれだけでサーバーが立ち上がります。シンプルですよね。

### HTMXとAlpine.js

このブログでは **HTMX** と **Alpine.js** も使っています。

- **HTMX** - HTMLの属性だけでAJAXリクエストを実現。SPAのようなUXを、シンプルに。
- **Alpine.js** - 軽量なリアクティブフレームワーク。ちょっとしたインタラクションに最適。

どちらも「必要最小限」を体現したライブラリで、この構成の哲学にぴったりです。
